## Características
- Gestión de canciones y biblias.
- Reproducción de videos con soporte de loops y backgrounds.
- Sistema de capas independiente (Texto / Medios / Blackout).
- Vista de escenario (Stage Display) con transiciones suaves.
- Control en tiempo real.

## Desarrollo
Proyecto construido con Electron, React, TypeScript y TailwindCSS.
